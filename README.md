## About Me

I'm an AWS Infrastructure Engineer at **Tesco Bank** with 7+ years of experience designing, building, and automating cloud infrastructure. My work spans IaC, CI/CD pipelines, monitoring, and platform reliability across enterprise-scale environments in financial services.

Most of my professional contributions live on private GitHub Enterprise accounts for **Tesco Bank** and **CirrusHQ** — so my public contribution graph doesn't tell the full story. I'm actively building out my public portfolio to better showcase what I work on day to day.
